## Roadmap Gaps

| Priorität | Lücke | Impact |
|---|---|---|
| 🔴 Hoch | Next.js **Page Router** — kein Backend-Handler | Viele bestehende Next.js-Projekte sind Page Router |
| 🔴 Hoch | **Vite / Express / Hono Backend-Adapter** — kein offizieller nicht-Next.js-Adapter | React SPA, Tanstack Start etc. nicht unterstützt |
| 🔴 Hoch | **`idp-drizzle`** — MySQL & SQLite fehlen | IDP-Feature für non-Postgres-Projekte nicht nutzbar |
| 🟡 Mittel | **Generischer OIDC-Consumer** — nur Google + GitHub hardcoded | Kein Keycloak, Auth0, Authentik etc. |
| 🟡 Mittel | **Edge-Runtime** — `@node-rs/argon2` + `rbac-yaml` (FS) blockieren | Cloudflare Workers, Vercel Edge Functions |
| 🟢 Niedrig | **Page Router tRPC** | Nischenanforderung |
