# Integrations

Integrations sit alongside plugins — they wire holeauth into adjacent ecosystems like tRPC, providing auth-aware context, transparent refresh rotation, and RBAC-gated procedures without modifying core behavior.

Unlike plugins which extend the auth instance itself, integrations provide helpers that bridge holeauth with other libraries and frameworks in your stack.
