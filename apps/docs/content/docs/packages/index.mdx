# Packages

Holeauth is distributed as a collection of npm packages. Core packages provide authentication primitives, framework adapters expose auth routes and server-helpers, React packages provide client components and hooks, and plugins add optional features like two-factor authentication, passkeys, role-based access control, and SSO.

## Plugin Drizzle adapters

These packages provide database adapters (table factories and CRUD helpers) for the plugin features above:

| Package | Purpose | Supported Databases |
| --- | --- | --- |
| `@holeauth/2fa-drizzle` | TOTP secrets, recovery codes | Postgres, MySQL, SQLite |
| `@holeauth/passkey-drizzle` | Passkey credentials | Postgres, MySQL, SQLite |
| `@holeauth/rbac-drizzle` | Groups, permissions, user-group assignments | Postgres, MySQL, SQLite |
| `@holeauth/idp-drizzle` | IDP client registry, authorization codes, refresh tokens | Postgres, MySQL, SQLite |
