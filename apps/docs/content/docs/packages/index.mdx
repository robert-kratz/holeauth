# Packages

holeauth is a small monorepo of focused packages. Install only what you need.

| Package | Install when |
| --- | --- |
| [`@holeauth/core`](/docs/packages/core) | Always — primitives for every other package. |
| [`@holeauth/nextjs`](/docs/packages/nextjs) | You are on Next.js App Router. |
| [`@holeauth/react`](/docs/packages/react) | You want `useSession` / `useAuth` hooks. |
| [`@holeauth/adapter-drizzle`](/docs/packages/adapter-drizzle) | You persist users & sessions with Drizzle ORM (pg / mysql / sqlite). |
| [`@holeauth/2fa-drizzle`](/docs/packages/2fa-drizzle) | Drizzle tables + adapter for `@holeauth/plugin-2fa`. |
| [`@holeauth/passkey-drizzle`](/docs/packages/passkey-drizzle) | Drizzle tables + adapter for `@holeauth/plugin-passkey`. |
| [`@holeauth/rbac-drizzle`](/docs/packages/rbac-drizzle) | Drizzle tables + adapter for `@holeauth/plugin-rbac`. |
| [`@holeauth/idp-drizzle`](/docs/packages/idp-drizzle) | Drizzle tables + adapter for `@holeauth/plugin-idp`. |
| [`@holeauth/plugin-2fa`](/docs/packages/plugin-2fa) | TOTP 2FA plugin with recovery codes. |
| [`@holeauth/plugin-passkey`](/docs/packages/plugin-passkey) | WebAuthn passkey plugin. |
| [`@holeauth/plugin-rbac`](/docs/packages/plugin-rbac) | RBAC plugin — groups, permissions, YAML-driven or dynamic. |
| [`@holeauth/plugin-idp`](/docs/packages/plugin-idp) | OIDC / OAuth 2.0 identity provider plugin. |
| [`@holeauth/rbac-yaml`](/docs/packages/rbac-yaml) | YAML loader + hot-reload for RBAC group definitions. |
