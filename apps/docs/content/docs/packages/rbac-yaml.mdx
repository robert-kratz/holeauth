# @holeauth/rbac-yaml

Node.js helper that loads, validates, and watches YAML-based RBAC group definitions for [`@holeauth/plugin-rbac`](/docs/packages/plugin-rbac). Resolves group inheritance, validates permission node syntax, and supports hot-reload in development.

<Callout type="info">
  This package is **Node.js only** — it uses `fs` and `path`. It does not run in edge runtimes. Call `loadRbacYaml` at server startup and pass the snapshot to the plugin.
</Callout>

## YAML file format

```yaml title="holeauth.rbac.yml"
groups:
  guest:
    default: true
    displayName: Guest
    permissions:
      - posts.read

  member:
    displayName: Member
    inherits:
      - guest
    permissions:
      - posts.create
      - posts.update

  editor:
    displayName: Editor
    inherits:
      - member
    permissions:
      - posts.publish
      - posts.delete

  admin:
    displayName: Admin
    permissions:
      - "*"
```

**Rules:**
- Exactly one group must have `default: true`. It is assigned to new users.
- `inherits` is resolved transitively (A → B → C merges all permissions).
- Permission nodes use dot-separated paths; `*` is a full wildcard; `!node` is a negation.
- Inheritance cycles throw at load time.

## `loadRbacYaml(filePath, options?)`

Load and parse a YAML file. Returns a `LoadedRbacYaml` handle with the parsed snapshot and reload support.

```ts title="lib/auth.ts"
import { loadRbacYaml } from '@holeauth/rbac-yaml';
import { rbac } from '@holeauth/plugin-rbac';

const yaml = loadRbacYaml('./holeauth.rbac.yml', {
  watch: process.env.NODE_ENV !== 'production',
});

const plugin = rbac({
  groups: yaml.snapshot.groups,
  adapter: rbacAdapter,
});

yaml.onReload((snapshot) => plugin.reload(snapshot.groups));
```

## Low-level utilities

### `buildSnapshot(raw, opts?)`

Build a `RbacConfigSnapshot` from a raw `RbacFile` object (already parsed YAML). Useful for testing or building custom loaders.

```ts
import { buildSnapshot } from '@holeauth/rbac-yaml';

const snapshot = buildSnapshot({ groups: { admin: { default: true, permissions: ['*'] } } });
```

### `resolveInheritance(raw)`

Resolve the transitive permission sets for all groups. Returns a `Map<string, string[]>` of group ID → effective permissions.

```ts
import { resolveInheritance } from '@holeauth/rbac-yaml';

const effective = resolveInheritance(rawFile);
// effective.get('editor') → ['posts.read', 'posts.create', 'posts.publish']
```

### `validateRbacDefinition(raw)`

Parse and validate a raw YAML object against the `RbacFileSchema` (Zod). Throws a `ZodError` if invalid.

```ts
import { RbacFileSchema } from '@holeauth/rbac-yaml';

const result = RbacFileSchema.safeParse(data);
if (!result.success) console.error(result.error.issues);
```

## Hot-reload in development

```ts title="lib/auth.ts"
const yaml = loadRbacYaml('./holeauth.rbac.yml', { watch: true });

yaml.onReload((snapshot) => {
  auth.plugins.rbac.reload(snapshot.groups);
  console.log(`[rbac] reloaded ${snapshot.groups.length} groups`);
});

// Clean up on process exit
process.on('SIGTERM', () => yaml.stop());
```
