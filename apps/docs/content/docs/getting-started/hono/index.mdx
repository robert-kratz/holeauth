# Hono

This guide walks you through integrating holeauth into a Hono application with Drizzle ORM. By the end, you will have email and password authentication, session management with automatic refresh-token rotation, and CSRF-protected endpoints. Hono's edge-native runtime makes it ideal for serverless deployments on Cloudflare Workers, Bun, Deno, or Node.js.

<Callout type="info">
  Holeauth is edge-compatible by design. The core auth runtime works seamlessly in Cloudflare Workers, Deno Deploy, and other edge environments. Database adapters depend on your chosen driver's edge support.
</Callout>

### Create database schema

Holeauth provides a table factory for each Drizzle dialect. Import the factory that matches your database, pass your existing users table, and export the resulting tables for use in migrations.

<Tabs items={['Postgres', 'MySQL', 'SQLite', 'Headless']}>
  <Tab value="Postgres">
```ts title="db/schema.ts"
import { pgTable, text, timestamp } from 'drizzle-orm/pg-core';
import { createHoleauthTables } from '@holeauth/adapter-drizzle/pg';

export const users = pgTable('user', {
  id: text('id').primaryKey().$defaultFn(() => crypto.randomUUID()),
  email: text('email').notNull().unique(),
  emailVerified: timestamp('email_verified', { withTimezone: true, mode: 'date' }),
  passwordHash: text('password_hash'),
  name: text('name'),
  image: text('image'),
  createdAt: timestamp('created_at', { withTimezone: true, mode: 'date' }).notNull().defaultNow(),
});

export const holeauth = createHoleauthTables({ usersTable: users });
```
</Tab>

### Create the auth instance

<Tabs items={['Postgres', 'MySQL', 'SQLite', 'Headless']}>
  <Tab value="Postgres">
```ts title="lib/auth.ts"
import { createHonoAuth } from '@holeauth/hono';
import { createHoleauthAdapters } from '@holeauth/adapter-drizzle/pg';
import { db } from '@/db/client';
import { holeauth } from '@/db/schema';

const adapters = createHoleauthAdapters({ db, tables: holeauth.tables });

export const auth = createHonoAuth({
  secrets: {
    jwtSecret: process.env.HOLEAUTH_SECRET!,
  },
  adapters: {
    user: adapters.user,
    session: adapters.session,
    account: adapters.account,
    auditLog: adapters.auditLog,
    verificationToken: adapters.verificationToken,
    transaction: adapters.transaction,
  },
  tokens: {
    accessTtl: 60 * 15, // 15 minutes
    refreshTtl: 60 * 60 * 24 * 30, // 30 days
    cookiePrefix: 'holeauth',
    sameSite: 'lax',
  },
  registration: {
    selfServe: true,
    inviteTtlSeconds: 7 * 24 * 60 * 60, // 7 days
    inviteUrl: ({ token }) =>
      `${process.env.APP_URL}/register/accept?token=${encodeURIComponent(token)}`,
  },
  allowDangerousEmailAccountLinking: false,
});
```

<Step>
### Mount the auth routes

Mount the holeauth sub-app at `/api/auth`:

```ts title="index.ts"
import { Hono } from 'hono';
import { auth } from './lib/auth';

const app = new Hono();

app.route('/api/auth', auth.app);

export default app;
```

This exposes the following routes:

- `POST /api/auth/register`
- `POST /api/auth/login`
- `POST /api/auth/logout`
- `POST /api/auth/refresh`
- `GET /api/auth/session`
- `POST /api/auth/change-password`
- `POST /api/auth/request-password-reset`
- `POST /api/auth/consume-password-reset`

See [Core API](/packages/core) for full request and response schemas.
</Step>

<Step>
### Add a protected route

Use the `getSession` helper to read the current session inside a Hono handler:

```ts title="index.ts" {3,10-19}
import { Hono } from 'hono';
import { auth } from './lib/auth';
import { getSession } from '@holeauth/hono';

const app = new Hono();

app.route('/api/auth', auth.app);

app.get('/api/me', async (c) => {
  const session = await getSession(c, auth);

  if (!session) {
    return c.json({ error: 'Unauthorized' }, 401);
  }

  return c.json({
    user: session.user,
  });
});

export default app;
```

Alternatively, use the middleware to populate `c.get('holeauthSession')` for all routes:

```ts title="index.ts" {3,8,12}
import { Hono } from 'hono';
import { auth } from './lib/auth';
import { holeauthHonoMiddleware } from '@holeauth/hono';

const app = new Hono();

app.route('/api/auth', auth.app);
app.use('*', holeauthHonoMiddleware(auth));

app.get('/api/me', (c) => {
  const session = c.get('holeauthSession');
  if (!session) return c.json({ error: 'Unauthorized' }, 401);
  return c.json({ user: session.user });
});

export default app;
```

<Callout type="info">
  Session validation happens automatically. If the access token is expired but the refresh token is valid, holeauth performs a transparent rotation and returns the fresh session.
</Callout>
