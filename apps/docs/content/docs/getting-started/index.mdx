# Getting started

Holeauth provides first-class integrations for multiple frameworks. Each guide walks you through setting up the core authentication system, then adding optional features like two-factor authentication, passkeys, role-based access control, and SSO.

<Callout type="info">
  Want your AI agent to scaffold the entire setup for you? Use the [`bootstrap-nextjs-holeauth`](/skills) or [`integrate-holeauth`](/skills) skills — they interview you and generate the boilerplate automatically. [Browse all skills →](/skills)
</Callout>

## Database adapters

<Callout type="info">
  Drizzle ORM is the default and recommended adapter. It supports Postgres, MySQL, and SQLite with full feature parity. If you need a different database or want complete control over data storage, you can implement the headless adapter interface described in [Adapters](/concepts/adapters).
</Callout>

Every framework guide offers tabs for Postgres, MySQL, SQLite, and Headless options so you can choose the adapter that fits your infrastructure.
