# Guides

Each guide is an end-to-end walkthrough — from configuration to a working UI.

- [Passkeys (WebAuthn)](/docs/guides/passkeys) — register and log in with platform authenticators.
- [Single Sign-On](/docs/guides/sso) — Google (OIDC + PKCE), GitHub (OAuth2), and custom providers.
- [Sessions & Refresh Rotation](/docs/guides/sessions-refresh) — the token lifecycle and reuse detection.
- [CSRF Protection](/docs/guides/csrf) — when to send the CSRF header, and why.
- [Events & Audit Logging](/docs/guides/events-audit) — observability and compliance-grade logs.
